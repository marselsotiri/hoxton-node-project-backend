
# We Chat

Here is the backend of the We Chat App.
It is created with Node.js, Prisma, Cors, Express.


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  ts-node index.ts
```
